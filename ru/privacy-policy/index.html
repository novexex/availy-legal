Политика конфиденциальности Availy

Дата вступления в силу: 9 июня 2026

Кратко

Availy — социальный календарь с фокусом на приватность. Приложение помогает друзьям планировать встречи через доступность и не показывает друзьям приватные детали календаря.

Какие данные собирает Availy

При использовании приложения Availy может собирать и хранить следующие данные:

  • Анонимный идентификатор аккаунта, созданный для установки приложения.
  • Информацию о друзьях, показанную в приложении: имена, usernames, инициалы и статус close friend.
  • Данные о доступности и расписании, включая временные окна, названия событий и категории событий, если они указаны.
  • Данные приглашений на встречи: временные окна, названия, заметки, места, статус и связанные ссылки на друзей.
  • Техническую информацию, необходимую для работы, защиты и диагностики приложения и backend-сервисов.

Как Availy использует данные

  • Чтобы показывать расписание, доступность, друзей и приглашения в приложении.
  • Чтобы разделять данные разных анонимных пользователей.
  • Чтобы поддерживать сценарии планирования встреч.
  • Чтобы диагностировать ошибки, защищать сервис и отвечать на обращения в поддержку.

Передача данных

Availy не продает персональные данные и не использует сторонние рекламные SDK. Друзья должны видеть только ту доступность и информацию о приглашениях, которую приложение предназначено показывать. Availy может использовать сервис-провайдеров, включая Supabase для backend и GitHub Pages для этого публичного сайта.

Хранение и безопасность

Availy хранит данные приложения в Supabase. Backend настроен так, чтобы изолировать строки данных по пользователям через row-level security. Ни один способ передачи или хранения данных не является абсолютно безопасным, но Availy построен так, чтобы ограничивать доступ к данным, необходимым для работы приложения.

Срок хранения и удаление

Availy хранит данные приложения, пока они нужны для предоставления сервиса. Если в приложении нет функции удаления, обратитесь в поддержку, и мы подскажем доступные варианты удаления данных.

Дети

Availy не предназначен для детей младше 13 лет. Если вы считаете, что ребенок передал данные через Availy, обратитесь в поддержку.

Изменения

Эта политика может обновляться по мере развития Availy. При существенных изменениях дата вступления в силу будет обновлена.

Контакты

По вопросам приватности или поддержки используйте страницу Поддержка Availy.
